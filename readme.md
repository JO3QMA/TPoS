# TPoS
## これは何？
The Terminal Poseter management Systemの略で、某案件用のポスター管理システムです。
名前すら決まってないのでOrgも作れないので、一旦個人用リポジトリで公開しております。

現在、`ruby`フォルダー内のもののみ動作します。それ以外は勉強しながらの開発となっており、動作するとは言えません。

# 何をするもの？
動作としては、複数の画像を結合するという簡単なものです。
